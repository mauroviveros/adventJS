| Command                                   | Action                                             |
| :---------------------------------------- | :------------------------------------------------- |
| `pnpm install`                            | Installs dependencies                              |
| `pnpm test`                               | Runs all tests in the project                      |
| `pnpm test editions/$YEAR`                | Runs tests for a specific edition **$YEAR**        |
